Privacy Policy - Gmail Filter: Show Only New Mail

Effective date: February 6, 2026

Gmail Filter: Show Only New Mail helps you focus by showing only Gmail messages that are new since your last session. This policy explains what data the extension accesses and how it is used.

Information We Access

  • Gmail metadata via the Gmail API (message and thread IDs, history IDs) to determine which messages are new.
  • Extension settings and session state (for example, whether the filter is enabled and the last history ID), stored locally in Chrome storage.

What We Do Not Access

  • We do not read or store email bodies or attachments.
  • We do not sell or share your data with third parties.
  • We do not transmit your data to our own servers.

How We Use Data

The Gmail metadata is used solely to identify newly received messages so the UI can filter your inbox. All processing happens locally in your browser.

Data Storage

The extension stores settings and the last known Gmail history ID in Chrome local storage. This data is used only to determine which messages are new.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
